Antes de abrir cualquier archivo de la carpeta, es necesario tener en cuenta lo siguiente:

1.- Este es un prototipo básico de página web. Esta solo tiene una estructura (HTML5) y un estilo definido (CSS3). Para
la elaboración de una página web más completa es necesario adquirir conocimientos acerca de Javacript para darle efectos
atractivos y dinámicos, así como también PHP para la comunicación con el servidor de la página y por último conocimientos
acerca de gestión de bases de datos como MySQL.

2.- Sumado a lo anterior, tampoco hemos adquirido los conocimientos para la elaboracion de una APP para telefono movil.

3.- Cuando te decidas a abrir el archivo de la página procura que este sea "index.html", ya que este es el archivo que contiene
el inicio de nuestra página. De todas formas, abrir cualquier archivo va a servir para navegar por la página, solo es
un consejo pensando en un escenario real de página web.

4.-El diseño NO ES ADAPTABLE, es decir no tiene un diseño responsivo (responsive web design) a distintos tamaños de la página, es 
por esto que SE TIENE que visualizar la página en PANTALLA COMPLETA. También se recomienda usar navegadores como Chrome u Opera 
para la visualización.


Por último, cabe destacar que esta página se trata de un prototipo, aún falta afinar y terminar algunos detalles.
